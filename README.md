# 🔐 Caesar Cipher Tool (Python)

A simple and beginner-friendly Python program that implements the Caesar Cipher algorithm to encrypt and decrypt messages.

---

## 🚀 Features
- 🔒 Encrypt text using Caesar Cipher
- 🔓 Decrypt encrypted messages
- 🔤 Supports uppercase and lowercase letters
- ✨ Keeps spaces, numbers, and symbols unchanged
- ⚡ Lightweight and easy to run

🧠 How It Works
The Caesar Cipher shifts each letter in the message by a fixed number (called the shift value).
Example:

HELLO → Shift 3 → KHOOR
📌 Example Usage

Enter your message: HELLO
Enter shift value: 3
Type 'e' for Encrypt

Output: KHOOR
